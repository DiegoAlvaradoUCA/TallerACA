# Guía de Instalación

Siga los pasos estructurados a continuación para inicializar el entorno de desarrollo local según su plataforma.

## Comandos de Inicialización

=== "Entorno Windows"
```bash
    pip install mkdocs-material
    mkdocs serve
    ```

=== "Entorno macOS / Linux"
```bash
    pip3 install mkdocs-material
    mkdocs serve
    ```

---

!!! note "Requisito Previo"
    Es necesario contar con Python 3.8 o una versión superior instalada en el sistema global antes de ejecutar los comandos de instalación.
